With this app you can present and scan QR code tickets (and a bit more) for UniNow courses.
It can be used for any digital booking system (DBS) hosted by them, e.g. for the ZHS München.

This app is <b>NOT</b> affiliated with UniNow GmbH, the provider/developer of the booking system.

Features:
<ul>
<li>Multiple sites and accounts</li>
<li>QR Code Check in (presenting and scanning)</li>
<li>List courses/appointments</li>
<li>Really fast, no ads, no trackers</li>
<li>Simple and small (~4MB)</li>
<li>Completely offline (authenticate once)</li>
</ul>
