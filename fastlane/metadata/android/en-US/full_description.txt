This android application is for doing the QR code check in (and a bit more) for UniNow courses.
It can be used for any booking system hosted by them, e.g. for the ZHS München.

This app is NOT affiliated with UniNow GmbH, the provider/developer of the booking system.


Features:
<ul>
<li>Multiple sites and accounts</li>
<li>QR Code Check in (presenting and scanning)</li>
<li>List courses/appointments</li>
<li>Really fast, no ads, no trackers</li>
<li>Simple and small (~7MB)</li>
<li>Completely offline (authenticate once)</li>
</ul>
